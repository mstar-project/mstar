1. Ad Hominem: This is a fallacy where the argument attacks the person making the argument rather than addressing the argument itself. For example, if someone says "global warming is a hoax because Al Gore believes in it," they are committing an ad hominem fallacy.

2. False Dilemma: This is a fallacy where only two options are presented as the only possible choices, when in reality there may be other options available. For example, if someone says "you're either with us or against us," they are presenting a false dilemma.

3. Slippery Slope: This is a fallacy where it is assumed that one event will inevitably lead to another, without any evidence to support this assumption. For example, if someone says "if we allow gay marriage, then people will start marrying animals," they are committing a slippery slope fallacy.
