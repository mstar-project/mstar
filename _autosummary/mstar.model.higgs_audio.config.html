mstar.model.higgs_audio.config#

Config for Higgs-Audio STT (bosonai/higgs-audio-v3-stt).

Higgs-Audio STT = Whisper-style audio encoder (“audio_tower”) + MLP projector (“audio_encoder_proj”) + dense Qwen3 text LLM. Values are read from the checkpoint’s config.json (top-level audio fields + text_config).
